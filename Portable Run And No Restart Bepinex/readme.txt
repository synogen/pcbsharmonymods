## Portable Run And No Restart Install for PC Building Simulator
Allows you to run programs directly from the USB stick by providing another tab "Portable".
Also makes installing and uninstalling programs not require a restart anymore, they immediately appear on the desktop/get removed again. 

## Requirements
Requires BepInEx 5.x for PC Building Simulator to be installed ( https://github.com/BepInEx/BepInEx/releases )

## Installation
Extract the mod folder "Portable Run And No Restart Install" to <PCBS folder>\BepInEx\plugins

## Compatibility
Should work with all other BepInEx plugins. Code changes in future versions of PC Building Simulator may break the mod.
