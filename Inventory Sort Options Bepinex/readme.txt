## Inventory Sort Options for PC Building Simulator
This mod gives you a dropdown with several sort options for your inventory. The options are:
Default, Newest First, Price Ascending, Price Descending, Name Ascending and Name Descending.

## Requirements
Requires BepInEx 5.x for PC Building Simulator to be installed ( https://github.com/BepInEx/BepInEx/releases )

## Installation
Extract the mod folder "Inventory Sort Options Bepinex" to <PCBS folder>\BepInEx\plugins

## Compatibility
Should work with all other BepInEx plugins. Code changes in future versions of PC Building Simulator may break the mod.
