## Show Detailed Inventory In Stock for PC Building Simulator
Shows new, used and broken item counts in the shop inventory display. For example, instead of just "Inventory: 3", it will show "☐3 (★1 ★1 ✝1)" -> 3 total, one new, one used, one broken.

Note: Items that are both used and broken are counted as broken only.

## Requirements
Requires BepInEx 5.x for PC Building Simulator to be installed ( https://github.com/BepInEx/BepInEx/releases )

## Installation
Extract the mod folder "Show Broken And Used Inventory In Shop" to <PCBS folder>\BepInEx\plugins

## Compatibility
Should work with all other BepInEx plugins. Code changes in future versions of PC Building Simulator may break the mod.
