## Faster 3D Mark for PC Building Simulator
Drops the 3D Mark duration to 9 seconds by default (3 seconds per stage).
If you want to customize the duration just open the "seconds_to_run.txt" in the mod directory and change the number of seconds you want the total duration to be in it, like 3 or 21. For an instant result set it to 0.

## Requirements
Requires BepInEx 5.x for PC Building Simulator to be installed ( https://github.com/BepInEx/BepInEx/releases )

## Installation
Extract the mod folder "Faster 3D Mark" to <PCBS folder>\BepInEx\plugins

## Compatibility
Should work with all other BepInEx plugins. Code changes in future versions of PC Building Simulator may break the mod.
